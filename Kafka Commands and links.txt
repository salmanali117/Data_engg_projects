To download JDK:
https://jdk.java.net/archive/

*Save to C:\Java\jdk11xyz..." and copy the path
*Create JAVA_HOME in system variables and give it the path of your jdk11xyz..
*Set %JAVA_HOME%\bin as a path

To install Kafka:
https://kafka.apache.org/downloads

*Scroll down to Kafka 3.7.0 
*Download the version compatible with scala 2.12 (important later)
*Extract the .tgz file to C:\Kafka

To start Kafka:
* Open terminal and navigate to C:\Kafka with the command "cd C:\Kafka"
* Start the zookeeper server with "bin\windows\zookeeper-server-start.bat config\zookeeper.properties"
* Open another tab in the terminal and navigate to C:\Kafka with the command "cd C:\Kafka"
* Start the Kafka server with "bin\windows\kafka-server-start.bat config\server.properties"
* Open another tab in the terminal and navigate to C:\Kafka with the command "cd C:\Kafka"
* Create a new topic with "bin\windows\kafka-topics.bat --create --topic test --bootstrap-server localhost:9092 --partitions 1 --replication-factor 1"
* Delete topic with "bin\windows\kafka-topics.bat --delete --topic test --bootstrap-server localhost:9092"
